#### 1) HTML 소개

웹에서 내용을 파악하기 쉽게 문서화하는 언어가 바로 HTML

Hyper Text Markup Language

- 웹페이지를 만드는 언어
- Hyper Text = 링크
- Markup Language = 마크업언어는 프로그래밍 언어의 한 종류 , 정보를 구조적으로 표현 가능
- 확장자가 html



#### 2) HTML문법 - 태그

- HTML 문법을 크게 6가지로 나누어 설명 예정 ( 태그, 속성, 중첩, 빈 태그, 공백 , 주석 )
- 태그 (TAG)
  - 꼬리표, 이름표 
  - ex : "<h1>Hello,HTML</h1>"
  - h1 : 태그 이름
  - 시작태그 , 내용 , 종료태그 (요소)
  - 태그는 HTML 뿐만 아니라 XML, SGML, XHML등 다양한 마크업 언어에서 모두 사용
